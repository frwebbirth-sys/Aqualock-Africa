How to Use

Setup guide and instructions

One-Time Setup (GitHub Auto-Save)

1
Go to GitHub Settings → Developer settings → Personal access tokens → Fine-grained tokens and create a new token with Contents: Read and write on your Aqualock repo only.
2
Go to the GitHub Settings tab in this admin panel. Enter your GitHub username, repo name, file path (portfolio.html), branch (main), and paste your token.
3
Click Test Connection to confirm everything works. You should see a green success message.
4
From now on, when you Add New Job and click Save & Publish, the job is automatically added to your portfolio.html on GitHub. GitHub Pages rebuilds and your site is live within ~30 seconds.
5
If GitHub is not configured, saving a job still works — it saves to your browser and you can use the 📋 Copy Code button to manually paste it into your file.
